# Security Policy

## Supported model

This repository is intended for defensive review of RDP bitmap cache artefacts on systems you are authorised to inspect.

## Third-party tooling

Third-party tools are pinned in `third_party.lock.json`. Do not update the pinned commits without reviewing the upstream diff and updating `SECURITY_REVIEW.md`.

## Reporting issues

Report security issues privately to the repository owner. Do not include RDP cache files, extracted screenshots, or reconstructed images in public issues.
